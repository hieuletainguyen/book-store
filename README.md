# Book store project

This is my first project using MySQL + Express Js + React JS + Node Js + Dockerfile + Docker compose to create a book store with these features:

- login/logout/sign-up
- Add book if you are an admin
- add books to cart

To start the project, run ```docker compose up -d``` in the main folder.

The book store app will be run on localhost:3000

The book store backend will be run on port 9897.

The database will be run on port 3306.

I have created docker compose to test the connection of the backend to database and frontend to backend separately. 

In order to test the connection of the backend to the database, I used postman to send the url.
